🏗️ CSS คืออะไร?

CSS (Cascading Style Sheets) ใช้ตกแต่งหน้าเว็บ — กำหนดสี, ขนาด, ตำแหน่ง, และทุกอย่างที่มองเห็น

🧱 โครงสร้างพื้นฐาน
selector {
  property: value;
}

/* ตัวอย่างจริง */
h1 {
  color: red;
  font-size: 24px;
  text-align: center;
}
📎 วิธีเชื่อม CSS กับ HTML
/* 1. Internal — ใน <style> */
<style>
  body { color: black; }
</style>

/* 2. External — ไฟล์ .css */
<link rel="stylesheet"
      href="style.css">

/* 3. Inline — ใน attribute */
<p style="color:red">
🎯 Selectors

Selector คือตัวเลือกว่า CSS จะไปตกแต่ง element ไหน

📌 ประเภท Selector พื้นฐาน
Selectorความหมายตัวอย่าง
*ทุก element* { margin:0 }
tagแท็ก HTMLp { color:red }
.classclass attribute.btn { ... }
#idid attribute#header { ... }
a, bทั้ง a และ bh1, h2 { ... }
a bb ภายใน adiv p { ... }
a > bb ลูกตรงของ aul > li { ... }
a + bb ถัดจาก a ทันทีh1 + p { ... }
a ~ bb พี่น้องหลัง ah1 ~ p { ... }
[attr]มี attribute นี้[type="text"]
🔬 Pseudo-class Selectors
a:hover { color: blue; } /* เมาส์ชี้ */
a:active { color: red; } /* กำลังกด */
a:visited { color: purple; } /* เคยคลิก */
:focus { outline: 2px solid blue; }

li:first-child { font-weight: bold; }
li:last-child { border: none; }
li:nth-child(2) { color: red; }
li:nth-child(odd) { background: #f0f0f0; }
li:nth-child(even) { background: white; }
:not(.disabled) { cursor: pointer; }

ย่อหน้าแรก → italic (first-child)

ย่อหน้าที่สอง → พื้นเหลือง (nth-child(2))

มี class .highlight → สีน้ำเงิน

ย่อหน้าสุดท้าย → underline (last-child)

ลองเอาเมาส์ชี้บรรทัดต่างๆ 👆

📦 Box Model

ทุก element ใน HTML คือกล่องสี่เหลี่ยม ประกอบด้วย 4 ชั้น

🗂️ ภาพ Box Model
margin (ระยะห่างนอก)
border (เส้นขอบ)
padding (ระยะห่างใน)
content
ข้อความ/รูป
📐 Properties ที่ใช้บ่อย
.box {
  /* ขนาด */
  width: 200px;
  height: 100px;
  max-width: 600px;

  /* padding */
  padding: 16px; /* ทุกด้าน */
  padding: 10px 20px; /* บน/ล่าง ซ้าย/ขวา */
  padding-top: 8px;

  /* border */
  border: 2px solid #333;
  border-radius: 8px;

  /* margin */
  margin: 0 auto; /* จัดกึ่งกลาง */

  /* สำคัญมาก! */
  box-sizing: border-box;
}
⚡ box-sizing: border-box คืออะไร?
❌ content-box (default)
width: 200px + padding: 20px
= ขนาดจริง 240px 😱

padding บวกเพิ่มเข้าไป ทำให้กล่องใหญ่กว่าที่กำหนด

✅ border-box (แนะนำ)
width: 200px + padding: 20px
= ขนาดจริง 200px ✓

padding อยู่ภายใน — ขนาดคือที่กำหนดเสมอ

/* Gradient */

background: radial-gradient(circle, #fff, #000);

📐 Flexbox

จัดวาง element ใน 1 มิติ (แถวหรือคอลัมน์) — ใช้บ่อยที่สุดใน CSS

🔧 Properties ของ Container
.container {
  display: flex;
  flex-direction: row; /* row|column */
  justify-content: center; /* แกนหลัก */
  align-items: center; /* แกนรอง */
  flex-wrap: wrap; /* ตัดบรรทัด */
  gap: 16px; /* ระยะห่างระหว่าง item */
}
.item {
  flex: 1; /* ขยายเต็มพื้นที่ */
  flex: 0 0 200px; /* grow shrink basis */
  align-self: flex-end; /* override align */
  order: 2; /* เรียงลำดับ */
}

🔲 CSS Grid

จัดวาง element ใน 2 มิติ (แถว + คอลัมน์) — สำหรับ layout ซับซ้อน

🔧 Properties
.grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  /* หรือย่อ: repeat(3, 1fr) */
  grid-template-rows: auto;
  gap: 16px;
}

/* Responsive อัตโนมัติ */
.auto-grid {
  grid-template-columns:
    repeat(auto-fit, minmax(200px, 1fr));
}

/* Item ยืดข้ามคอลัมน์ */
.wide { grid-column: span 2; }
.full { grid-column: 1 / -1; }

📍 Position

ควบคุมตำแหน่งของ element นอกเหนือจาก flow ปกติ

📋 ประเภท Position
ค่าความหมาย
staticค่าเริ่มต้น — ตาม flow ปกติ, top/left ไม่มีผล
relativeเลื่อนจากตำแหน่งปกติตัวเอง
absoluteยึดกับ parent ที่มี position ≠ static
fixedยึดกับ viewport — ไม่เลื่อนตาม scroll
stickyตาม scroll จนถึงจุดที่กำหนด แล้วค้าง
.parent { position: relative; }
.child {
  position: absolute;
  top: 0; right: 0;
  z-index: 10; /* ชั้นซ้อน */
}
🎮 ตัวอย่าง position: absolute
relative
top:10 left:20
absolute
bottom:8 right:8

🟦 relative: เลื่อนจากตำแหน่งปกติ  |  🟩 absolute: ยึดมุม parent

/* จัดกึ่งกลางด้วย absolute */
.center {
  position: absolute;
  top: 50%; left: 50%;
  transform: translate(-50%, -50%);
}
🌊 Transitions

เพิ่ม animation ให้กับการเปลี่ยนแปลง state เช่น hover

🔧 Syntax
transition: property duration timing-function delay;

/* ตัวอย่าง */
transition: all .3s ease;
transition: color .2s, background .5s;

/* timing functions */
transition-timing-function: ease; /* เริ่มช้า-เร็ว-ช้า */
transition-timing-function: linear; /* สม่ำเสมอ */
transition-timing-function: ease-in; /* เริ่มช้า */
transition-timing-function: ease-out; /* จบช้า */
transition-timing-function: cubic-bezier(.17,.67,.83,.67);
🎮 ลองชี้เมาส์ที่ปุ่ม
.btn {
  transition: all .3s ease;
}
.btn:hover {
  transform: translateY(-2px);
  box-shadow: 0 8px 20px rgba(0,0,0,.2);
}
✨ Animations (@keyframes)

สร้าง animation ที่ทำงานอัตโนมัติโดยไม่ต้องมีการโต้ตอบ

🔧 Syntax
/* 1. สร้าง keyframes */
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-20px); }
}

/* 2. ใช้กับ element */
.bouncing {
  animation-name: bounce;
  animation-duration: 1s;
  animation-iteration-count: infinite;
  animation-timing-function: ease;
  /* ย่อ: */
  animation: bounce 1s ease infinite;
}

🔮 Pseudo-elements

  display: block;
  position: absolute;
}

.element::after {
  content: "";
  width: 100%;
  height: 2px;

::selection {

📱 Responsive Design

ทำให้หน้าเว็บแสดงผลได้ดีทุกขนาดหน้าจอ

📏 Media Queries
/* Mobile First (แนะนำ) */
.card { width: 100%; }

@media (min-width: 640px) {
  .card { width: 50%; }
}
@media (min-width: 1024px) {
  .card { width: 33.33%; }
}

/* Breakpoints ที่ใช้บ่อย */
sm: 640px
md: 768px
lg: 1024px
xl: 1280px

/* หน้าจอแนวนอน */
@media (orientation: landscape) { ... }

/* dark mode */
@media (prefers-color-scheme: dark) { ... }
📐 Units ที่ใช้กับ Responsive

หน่วยต่างๆ เทียบกัน:

100% — เต็มพื้นที่ parent
60vw — 60% ของ viewport
2rem = 32px (ถ้า root 16px)
1em = font-size ปัจจุบัน
clamp(120px, 30%, 300px)
/* clamp(min, preferred, max) */
font-size: clamp(1rem, 2.5vw, 2rem);
width: clamp(200px, 50%, 800px);

/* อย่าลืม meta viewport ใน HTML! */
<meta name="viewport"
      content="width=device-width, initial-scale=1">
กล่องนี้เปลี่ยนสีตาม breakpoint
🔧 CSS Variables (Custom Properties)

เก็บค่าที่ใช้ซ้ำ — เปลี่ยนที่เดียว ส่งผลทุกที่

📝 วิธีใช้
/* ประกาศใน :root (global) */
:root {
  --primary: #6366f1;
  --spacing: 16px;
  --radius: 8px;
  --shadow: 0 4px 12px rgba(0,0,0,.1);
}

/* ใช้งาน */
.btn {
  background: var(--primary);
  padding: var(--spacing);
  border-radius: var(--radius);
  box-shadow: var(--shadow);
}

/* Override ใน component */
.danger { --primary: #ef4444; }

/* fallback value */
color: var(--text, black);
🎨 ตัวอย่าง Theme
ธีม Purple: var(--theme) = #8b5cf6
ธีม Orange: var(--theme) = #f59e0b
.card {
  --theme: #8b5cf6;
  background: var(--theme);
}
.card.orange {
  --theme: #f59e0b; /* override! */
}

💡 ทำไมต้องใช้ CSS Variables?

  • เปลี่ยน theme ได้ทั้งไซต์ด้วยโค้ดบรรทัดเดียว
  • JavaScript สามารถอ่านและเปลี่ยนค่าได้
  • Scope ได้ — ต่างจาก Sass variables
  • Dynamic — คำนวณใหม่เมื่อค่าเปลี่ยน

p { color: red; } /* 0-0-1 */
